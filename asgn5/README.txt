Three.js Campfire Scene Assignment

How to run:
1. Open this folder in VS Code.
2. Install/use the Live Server extension OR run: python3 -m http.server 8000
3. Open http://localhost:8000 in your browser.

Do not double-click index.html because module imports and model loading work best through a local server.

Rubric checklist:
- 20+ primary shapes: ground, stones, logs, flames, trees, moon, fireflies, bench pieces.
- Three different shape kinds: cylinder, sphere, cone, box, dodecahedron.
- Textured object: ground and logs.
- Animated object: flames, moon, fireflies, flickering point light.
- Textured GLTF model: DamagedHelmet.gltf loaded with GLTFLoader.
- 3+ light types: AmbientLight, DirectionalLight, PointLight, SpotLight.
- Textured skybox: CubeTextureLoader background.
- Perspective camera: PerspectiveCamera.
- Mouse controls: OrbitControls.
- Wow point: interactive campfire scene with animated fireflies/flames and visible note on page.
