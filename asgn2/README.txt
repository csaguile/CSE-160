How to run:
1. Unzip this folder.
2. Open index.html in Chrome.
3. If browser blocks local files, run local server:
   python3 -m http.server 8000
   then open http://localhost:8000

Files included:
- index.html
- blockyanimal.js
- Cube.js
- Cylinder.js
- lib/webgl-utils.js
- lib/cuon-utils.js
- lib/cuon-matrix.js

Controls:
- Drag canvas to rotate.
- Shift-click canvas for poke animation.
- Use buttons/sliders for leg, neck, tail animations.
